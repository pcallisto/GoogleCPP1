Lots of commented functions related to : 
	https://developers.google.com/edu/c++/getting-started

but right not set to run a times table to a text file.
Shows use or several C++11 methods and features like:
stringstream, string method's, stoi, sprintf_s, ofstream, I/O manipulators 
like: ::right, ::setfill, ::setw . Also, a few function templates thrown in
for good measure. 
T digitCount(T arg) returns the number of digits in a number
T numeralValue(T arg) returns the Numerology value of received value to 
												one digit using recursion
